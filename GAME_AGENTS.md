# GAME_AGENTS.md — 게임 도메인 지식

이 파일은 AI 에이전트가 개발 과정에서 학습한 패턴, 함정, 아키텍처 결정을 누적하는 곳입니다.
각 개발 사이클에서 에이전트가 자동으로 업데이트합니다.

---

## 학습된 패턴 (Learned Patterns)

> 아직 학습된 패턴이 없습니다. 개발 진행 중 에이전트가 자동으로 기록합니다.

형식:
```
### [패턴 제목]
- **상황**: [어떤 문제에 직면했는가]
- **해결**: [무엇을 했는가]
- **적용 대상**: [어떤 시스템/파일에]
- **참고**: [관련 이슈/커밋]
```

---

## 피해야 할 함정 (Pitfalls to Avoid)

> 아직 기록된 함정이 없습니다. GDScript 관련 일반적 함정은 CLAUDE.md를 참조하세요.

형식:
```
### [함정 제목]
- **문제**: [무엇이 잘못되는가]
- **원인**: [왜 발생하는가]
- **해결책**: [올바른 접근법]
- **참고**: [README 섹션 또는 외부 문서]
```

---

## 시스템 아키텍처 결정 (Architecture Decisions)

> 아직 기록된 결정이 없습니다. 첫 아키텍처 결정은 Phase 2 코어 엔진 설계 시 기록됩니다.

형식:
```
### [결정 제목]
- **결정**: [무엇을 결정했는가]
- **대안**: [고려했던 다른 선택지]
- **이유**: [왜 이 결정을 했는가]
- **영향**: [어떤 시스템에 영향을 미치는가]
- **날짜**: [YYYY-MM-DD]
```
